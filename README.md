# Lights, Beats, Action!
## "Lights, Beats, Action!" - ir aizraujoša darbības ritma datorspēle, kurā enerģiskas mūzikas ritmā jāizvairās no šķēršļiem, lai sasniegtu uzvaru, un katrs var ne tikai spēlēt jau izveidotus "bāzes" līmeņus, bet arī izveidot un spēlēt savus vai citu spēlētāju līmeņus.  
## Izmantotas Tehnoloģijas
- Unity Engine / Editor 6000.2.10f1;
- .NET 8;
- C# 12;
- Supabase PostgreSQL 17.6
